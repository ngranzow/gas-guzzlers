# gas-guzzlers


## Description
This is a website that build to track the commute and gas price.  <br>


## Table of Contents
* [Installation](#installation)
* [Usage](#usage)
* [Contributing](#contributing)

## Installation
no installation need, please visit https://mysterious-plateau-73573.herokuapp.com/ and enjoy!  <br>

When you visit our page first time,
You will see the homepage this below, along with a chart that show the previous data  <br>

you may login via here , or, you may signup one <br>
